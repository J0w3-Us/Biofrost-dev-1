# Curso de VSCode
Un repositorio donde aprenderemos un par de trucos para trabajar rápidamente con Visual Studio Code.

También hay un par de extensiones bastante útiles que les recomiendo de entrada

* [Descargar VSCode - Sitio Oficial](https://code.visualstudio.com/)

## Extensiones

# Indispensables:

* [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens) - Muestra errores directamente en el código
* [Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme) - Iconos bonitos para archivos y carpetas

# Desarrollo:

* [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) - Asistente de IA para programación
* [GitHub Copilot Chat](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat) - Chat integrado con Copilot
* [Code Runner](https://marketplace.visualstudio.com/items?itemName=formulahendry.code-runner) - Ejecuta código rápidamente
* [IntelliCode](https://marketplace.visualstudio.com/items?itemName=VisualStudioExptTeam.vscodeintellicode) - Sugerencias inteligentes de código

# Python:

* [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python) - Soporte completo para Python
* [Pylance](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance) - Análisis de código Python
* [Python Debugger](https://marketplace.visualstudio.com/items?itemName=ms-python.debugpy) - Depurador para Python
* [Python Snippets](https://marketplace.visualstudio.com/items?itemName=ericsia.pythonsnippets3) - Snippets útiles para Python

# C#/.NET:

* [C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit) - Kit completo para desarrollo C#
* [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) - Soporte para C#
* [IntelliCode for C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.vscodeintellicode-csharp) - Sugerencias inteligentes para C#

# Web Development:

* [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) - Formateador de código
* [HTML CSS Support](https://marketplace.visualstudio.com/items?itemName=ecmel.vscode-html-css) - Soporte mejorado para HTML/CSS
* [npm Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.npm-intellisense) - Autocompletado para módulos npm
* [HTML5 Boilerplate](https://marketplace.visualstudio.com/items?itemName=sidthesloth.html5-boilerplate) - Plantillas HTML5
* [Prisma](https://marketplace.visualstudio.com/items?itemName=Prisma.prisma) - Soporte para Prisma ORM

# Apariencia y Visual:

* [Fluent Icons](https://marketplace.visualstudio.com/items?itemName=miguelsolorio.fluent-icons) - Iconos con estilo Fluent
* [VSCode Animations](https://marketplace.visualstudio.com/items?itemName=brandonkirbyson.vscode-animations) - Animaciones para VS Code
* [Custom CSS and JS Loader](https://marketplace.visualstudio.com/items?itemName=be5invis.vscode-custom-css) - Personalización avanzada de CSS
* [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow) - Colores para indentación
* [Rainbow Brackets](https://marketplace.visualstudio.com/items?itemName=tal7aouy.rainbow-bracket) - Paréntesis de colores

# GitHub & Remote:

* [GitHub Codespaces](https://marketplace.visualstudio.com/items?itemName=GitHub.codespaces) - Desarrollo en la nube
